# Operator Configuration Template

Copy this file to `.local/config.md` and fill in your values.

The `.local/` directory is gitignored. Do not treat the populated file as a normal agent input.

This template may hold a mix of operator-mediated facts and never-read sensitive local values. Because those categories can live side by side, agents must not read the populated file directly. If an AI agent needs a runtime fact, disclose only the minimum required non-sensitive value or run the command locally and relay the result.
