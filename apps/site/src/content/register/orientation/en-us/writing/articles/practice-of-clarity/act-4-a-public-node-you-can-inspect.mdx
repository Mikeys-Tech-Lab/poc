### What this changes

Instead of only reading finished text, you can inspect:

- how the current structure and decisions came to be
- changes over time
- what was uncertain when decisions were made
- what was not fully resolved

That matters because it shows not just what was decided, but how those decisions were formed.

### What the repository is

The repository is a working example of the practice in use.

It is not the whole system. It is a visible, bounded node.

Inside the repository, the guided entry paths are `onboard me` and `Evolution Arc`.

If you want a manual reading path, start with the [onboarding guide](https://github.com/Mikeys-Tech-Lab/poc/blob/main/docs/onboarding/manual.md) or [Evolution Arc](https://github.com/Mikeys-Tech-Lab/poc/blob/main/docs/onboarding/evolution-arc.md).

You can inspect the public node directly in the [repository](https://github.com/Mikeys-Tech-Lab/poc).
