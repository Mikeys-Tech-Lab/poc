This site is a **publication surface** for the Practice of Clarity.

It is designed for long-form reading: essays and supporting pages that make reasoning visible so shared understanding can form without pretending certainty.

## Reading this site

The site adapts to how you want to read through three controls.

### Register

The site title shows the current register: **Practitioner** or **Orientation**. Click the title to switch.

Practitioner assumes comfort with technical and organisational language. Orientation presents the same structure and ideas in more accessible language, with gentler pacing and clearer examples. Nothing is hidden. Only the voice changes.

### Theme

The theme selector lets you choose a visual style. Options include atmospheric and solid in both dark and light variants. This changes readability and preference, not meaning.

The accessibility button opens a reading panel with additional controls for comfort and access.

It currently includes:

- **Increase contrast**
- **Text size**
- **Reduce motion**
- **Solid background for higher contrast**
- **Underline links**
- **Alternate font**
- **Reset to defaults**

These controls change presentation, not content. They exist to support different cognitive, visual, and sensory needs without creating a separate version of the site.

## What you can expect

This is a **small launch surface**.

It currently includes:

- core framing pages
- the first essay in Practice of Clarity
- public license pages
- visible infrastructure and development transparency

The repository already exists. This site simply keeps the writing in the foreground while the practice is introduced in sequence.

## What this is not

This is not:

- surveillance
- forced self-documentation
- performative productivity
- a requirement to perform certainty
- transparency demanded downward without reciprocal accountability

Making reasoning visible only has value if it reduces confusion, shares context, and makes responsibility easier to locate. If it becomes another way to extract labor or expose people without protection, it has failed the practice.

## Where to start

Start with [Act I: When Output Outpaces Understanding](/en-us/writing/articles/practice-of-clarity/act-1-when-output-outpaces-understanding/), the opening essay in this publication.
