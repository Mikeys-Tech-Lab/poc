This site is a **publication surface** for the Practice of Clarity.

It is designed for long-form reading — essays, lenses, and bridges — not repository documentation.

The practice itself is simple: **make reasoning visible** so shared understanding can form without pretending certainty.

## How to use this site

There are three controls in the top-right (and in the mobile sidebar):

- **Language**: switch between locale versions of the pages (en‑US, en‑GB, de‑DE).
- **Register**: switch between **Practitioner** and **Beginner** writing.
  - Practitioner assumes comfort with technical and organisational language.
  - Beginner keeps the same ideas but uses everyday language and gentler examples.
- **Theme**: change how the site looks. This does not change meaning — it changes readability and preference.

## What you can expect

- This is a **living publication**. Some pages are placeholders while the structure is validated.
- The practice is published in a sequence (see the [publication arc](/en-us/about/publication-arc/)).
- The repository and implementation details are intentionally published later.

## Where to start

- If you want the release sequence, read [publication arc](/en-us/about/publication-arc/).
- If you want a first essay, open the [placeholder article](/en-us/articles/placeholder/) (to validate format and navigation).
