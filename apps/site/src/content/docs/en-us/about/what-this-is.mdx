This site is a **publication surface** for the Practice of Clarity.

The practice itself is simple: **make reasoning visible** so shared understanding can form without pretending certainty.

## Reading this site

The site adapts to your preferences through two controls.

### Register

The site title shows the current register: **Practitioner** or **Orientation**. Click the title to switch.

Practitioner writing assumes comfort with technical and organisational language. Orientation writing covers the same ideas using everyday language and gentler examples. Both registers contain the same structure. Nothing is hidden. Only the voice changes.

### Theme

The theme selector lets you choose a visual style. Options include atmospheric (blurred background) and solid (opaque) in both dark and light variants. This changes readability and preference, not meaning.

## What you can expect

This is a **small launch surface**. It currently includes the framing pages, the first writing route, and the licence pages. The repository already exists; this site simply keeps the writing in the foreground.

## Where to start

Start with [Act I — When Output Outpaces Understanding](/en-us/writing/articles/practice-of-clarity/act-1-when-output-outpaces-understanding/), the opening writing route in this publication.
