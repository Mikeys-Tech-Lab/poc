This site is a **publication surface** for the Practice of Clarity.

It is designed for long-form reading: essays, lenses, and bridges. The practice itself is simple: **make reasoning visible** so shared understanding can form without pretending certainty.

## Reading this site

The site adapts to your preferences through three controls.

### Register

The site title shows the current register: **Practitioner** or **Orientation**. Click the title to switch.

Practitioner writing assumes comfort with technical and organisational language. Orientation writing covers the same ideas using everyday language and gentler examples. Both registers contain the same structure. Nothing is hidden. Only the voice changes.

### Language

The language selector in the header (and in the mobile sidebar) lets you switch between available locales: English (US), English (UK), and Deutsch. Each locale is a full translation, not a machine-generated copy.

### Theme

The theme selector lets you choose a visual style. Options include atmospheric (blurred background) and solid (opaque) in both dark and light variants. This changes readability and preference, not meaning.

## What you can expect

This is a **living publication**. Some pages are placeholders while the structure is validated. The practice is published in a deliberate sequence (see the [publication arc](/about/publication-arc/)). The repository and implementation details are intentionally published later.

## Where to start

If you want the release sequence, read [publication arc](/about/publication-arc/). If you want a first essay, open the [placeholder article](/articles/placeholder/) to see the format and navigation.
