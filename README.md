# Practice of Clarity

A living practice of clarity designed for federation. Not a framework. Not a method. A set of seeds that grow differently in every context.

## Get started

If you are using an AI-assisted editor (Cursor, Claude Code, GitHub Copilot), say **"onboard me"** in chat. The agent will ask what depth you need and walk you through the relevant topics.

For a manual path without AI, read the [onboarding guide](docs/onboarding/manual.md).

## Quick commands

```bash
pnpm install
pnpm run local        # docs site at http://localhost:4321
pnpm run build        # production build
pnpm test             # tooling tests
```

Full setup details: [docs/onboarding/local-setup.md](docs/onboarding/local-setup.md)
