This package is the canonical home for the `Sensible Defaults`
mandate lens in this repo.

It is based on earlier lineage, but the live package surface now resides here.

## Files

- `lens.md` - canonical application artifact
- `context-seeder.md` - derivative portable delivery form

## Intended use

Use this lens when a delivery context is drowning in hidden work, planning
compression, cognitive overload, or timeline optimism.

Its job is not to generate exhaustive process.

Its job is to surface the missing delivery reality early enough to reduce
pressure and rework.

## Activation boundary

Reading about the lens is not the same as activating it.

Operational grounding begins only when the seeder and canonical artifacts are
loaded.

This package is intended for reasoning-capable language models, thinking-capable
agents, IDE agents, and comparable execution environments.

It is not designed for blind training, scraping, or passive ingestion.
